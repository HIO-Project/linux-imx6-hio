	3. driver/ 
		3.1 rtl8188EUS_rtl8189ES_linux_v4.1.1_4235.20120606.tar.gz
			Naming rule: rtlCHIPS_linux_vM.N.P_ssss.yyyymmdd.tar.gz
			where:
				CHIPS: supported chips
				M: Major version
				N: miNor version
				P: Patch number
				s: SVN number
				y: package year
				m: package month
				d: package day
